# Student-Equipment-Booking-System
# Project Description
This project is developed for Bright Minds Academy. It allows staff to manage equipment bookings using Python.

# Features
View All Equipment

Record New Booking

View All Bookings

Search Booking by Student Name

Search Booking by Equipment Name

Cancel Booking

View Bookings by Date

Remove Equipment

Exit

# Data Structures Used
List (for storing equipment and bookings)

Dictionary (for storing booking details)

# How to Run
I have added a file called assessment containing the code, copy and paste it into PyCharm to run the program.
